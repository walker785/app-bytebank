# Bytebank app

Aplicativo desenvolvido durante o curso "Fundamentos de Flutter: Crie o seu primeiro App" fornecido por Alura Cursos Online, cujo o objetivo é desenvolver um aplicativo simples que realiza transferências e as cadastra em lista.
